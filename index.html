Key Vocabulary from Unit 1 & 2

  • Bit: A binary digit, either 0 or 1.
  • Byte: A sequence of 8 bits.
  • Roundoff: Error when there arenâ€™t enough bits to represent a number precisely.
  • Analog Data: Values that change smoothly over time.
  • Lossless: Compression that preserves all original data.
  • Lossy: Compression that removes some data permanently.
  • Metadata: Data about data.
  • Computing Device: A device that can run a program (computer, smartphone).
  • Computer Network: Interconnected devices that send/receive data.
  • Path: The series of connections between devices.
  • Protocol: Rules for how systems communicate.
  • Fault Tolerant: Ability of a system to keep working if parts fail.
  • Scalability: Ability of a system to grow/shrink as needed.
  • Bandwidth: Maximum transmission capacity, measured in bit rate.
  • URL: An easy-to-remember web address.
  • IP (Internet Protocol): Assigns addresses and routes data.
  • TCP: Reliable, ordered, error-checked delivery.
  • UDP: Faster, lightweight delivery with less reliability.
  • Packets: Small chunks of data sent across networks.
  • Routers: Devices that forward data across networks.
  • DNS: Translates domain names into IP addresses.
  • HTTP: Protocol for web pages/media.
  • WWW: System of linked web pages and files browsable online.
