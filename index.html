Unit 1 & Unit 2 Vocabulary

Unit 1: Data is represented in bits and bytes. Metadata describes data, while lossless and lossy compression determine how data is stored. Analog data can be converted to digital, and issues like roundoff appear when precision is limited.

Unit 2: A computing device connected to a computer network communicates using protocols. The internet is fault tolerant and scalable, with routers choosing the best path for packets. Tools like DNS, IP, TCP, and HTTP make the World Wide Web possible.

Cloud Computing

Cloud computing is the delivery of computing services—such as storage, processing power, software, and databases—over the internet instead of relying on a single local computer. It allows people and organizations to access data and applications from anywhere, on any device, as long as they have an internet connection. Cloud computing is scalable, meaning resources can grow or shrink as needed, and it uses powerful networks and protocols to ensure reliability and efficiency. This technology supports everything from everyday apps to advanced data science, making it a key tool for modern innovation.
